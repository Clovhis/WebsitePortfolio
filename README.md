# WebsitePortfolio (GitHub Pages)

Minimal, professional portfolio site for a DevOps / SRE profile.

## Publish (GitHub Pages)
1. Go to the repo **Settings**.
2. Open **Pages**.
3. Under **Build and deployment**:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Save. After it builds, the site will be available at:
   - `https://clovhis.github.io/WebsitePortfolio/`

## Update Content
- Main page: `index.html`
- Styles: `assets/css/site.css`
- JS (logos marquee + reveal): `assets/js/main.js`
- Photo: `assets/img/photo.jpg`
- CV: `assets/cv/Leonardo_Vargas_CV_2026.pdf`
